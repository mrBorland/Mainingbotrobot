# OKXFarmBot

Telegram-бот для автоматизованого фарму airdrop-завдань на платформах:
- Galxe
- OKX
- TaskOn

## Функції:
- Автоматичне виконання фарму
- Перевірка балансу USDT
- Вивід коштів з акаунтів OKX
- Telegram-інтерфейс з кнопками /farm, /balance, /withdraw
